使用KMS激活Windows10或者Office系列软件

发表于 2018-04-25 | 分类于 Windows相关教程

Attention！！！
使用该软件激活Windows或者Offic必须退出电脑的安全软件，例如360安全卫士，腾讯电脑管家,Windows Defender或者电脑自带的安全软件/杀毒软件，一般的软件退出就直接右键退出，WindowsDefender可能比较难关闭，可以参考这篇教程，这些软件会阻止该软件的运行，甚至会提示该软件是病毒，不过可以放心，这个软件是无毒的
首先给出这个软件的 下载地址 密码：vbl8

1.下载KMS，双击安装

2.点击Next

3.协议选择界面勾选我同意选项，点击Next

4.选择安装路径(保持默认就行)，点击Next

5.设置开始菜单文件夹，依然不管，点击Next，然后就会自动安装，安装完成之后安装界面会自动消失

6.点击windows图标打开开始菜单

7.找到AutoPico或者KmsPico这个软件(我这里只出现AutoPico，AutoPico是自动运行的，如果是选择KmsPico那么等它软件界面出来之后点击红色按钮就行)在它上面右键然后选择更多然后选择以管理员身份运行


8.当听到Program Complete的时候，windows或者office就已经激活了
