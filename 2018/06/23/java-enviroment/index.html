Windows下Android开发环境配置(Win10)

发表于 2018-06-23 | 分类于 Windows相关教程

在Windows上搭建Java(Android)开发环境 –ADT Bundle

前言:本篇是基于ADT Bundle(X86/X64)所写，如果电脑配置较好，或者有一定基础开发经验之后可以使用Android Studio，下一篇介绍Android Studio的安装

官网下载JDK, 要分清是下载32位还是64位的版本。

地址: 点我进入下载

安装，这里就不演示了，下载下来之后安装JDK，一直点下一步就是，安装路径最好不要修改

配置环境变量，桌面右键此电脑图标(桌面没有图标也可按Windows键+E进入资源管理器然后在空白的地方右键点击属性)，点击属性。

点击高级系统设置

找到环境变量

找到系统变量

点击Path

点击编辑 填入JDK安装下的bin目录,点击确定，我的是64位的默认路径是

C:\Program Files\Java\jdk1.8.0_171\bin
如果你下载的是32位的那就是
C:\Program Files(x86)\Java\jdk1.8.0_171\bin

在系统变量界面新建JAVA_HOME变量，名称填入JAVA_HOME,变量值填入安装路径，点确定

在系统变量界面新建变量，名称填入CLASSPATH,变量值填入安装路径下的lib目录，点确定

检测是否安装成功

按Windows+R，输入cmd，输入

java -version
如果是类似如下界面,则安装成功，如果不是请根据上面几个步骤检查，或者重新按步骤来
